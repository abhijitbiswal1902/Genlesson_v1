# **App Name**: LessonGen

## Core Features:

- Topic Input: Accept user input for the lesson topic using a textbox.
- Lesson Generation: Call the Gemini API using fetch to generate a lesson based on the topic provided by the user.  API key must be set.
- Lesson Display: Display the generated lesson on the frontend below the input box in a readable format.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) -> RGB(#3399FF) - A bright, clear blue for learning.
- Background color: HSL(210, 20%, 95%) -> RGB(#F0F5FA) - Light, desaturated blue background.
- Accent color: HSL(180, 70%, 40%) -> RGB(#33CCCC) - A contrasting teal to highlight interactive elements.
- Font: 'Inter', sans-serif - Suitable for both headlines and body text, ensures readability and a modern feel.
- Centered layout with minimal styling.
- Subtle fade-in animation for the generated lesson text.
